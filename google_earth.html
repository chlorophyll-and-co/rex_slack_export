Archive Unused Layer Data

2017-08-06 23:40
In Google Earth Pro, you can archive a sub-folder or any element in your project to .kmz. This can be useful if you?ve imported high resolution (large) shapefiles (like contour lines) to help determine ridges and valley, but no longer need them. To archive, select the sub-folder you want to archive, then select ?Save Place As?? from the File menu. If you?re using the REX Keyline Layers in your local file system, save it in the appropriate directory (contours go in 2_Geography). Once the .kmz file is saved, you can delete the layer. If you need it later, you can reload it from the .kmz. **Note: this saved my 90MB! Which was 2/3 of my total project data.
